C++ explicit关键字

By kenithcai

2016-07-29 更新日期:2016-07-29

explicit作用:

在C++中，explicit关键字用来修饰类的构造函数，被修饰的构造函数的类，不能发生相应的隐式类型转换，只能以显示的方式进行类型转换。

explicit使用注意事项:

* explicit 关键字只能用于类内部的构造函数声明上。

* explicit 关键字作用于单个参数的构造函数。

* 在C++中，explicit关键字用来修饰类的构造函数，被修饰的构造    
 函数的类，不能发生相应的隐式类型转换

例子：

未加explicit时的隐式类型转换


class Foo  
{  
public:  
     Foo(string str):m_str(str){}

private:
    string m_str;
};

调Foo创建的时候


int main(int argc, char* argv[])
{
    //发生隐式类型转换  
      //编译器会将它变成如下代码  
      //tmp = Foo("str")  
      //Foo A(tmp);  
      //tmp.~Foo(); 
      Foo A = "str";
}

加了explicit关键字后，可防止以上隐式类型转换发生


class Foo  
{  
public:  
     explicit  Foo(string str):m_str(str){}

private:
    string m_str;
};


Foo A = "str";    // 会报错
Foo B = A;        // 也会报错

只能 
Foo A = Foo("aa");
