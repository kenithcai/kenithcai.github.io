学习opengl

By kenithcai

搭建windows下面的环境

GLUT代表OpenGL应用工具包，英文全称为OpenGL Utility Toolkit，是一个和窗口系统无关的软件包，它由Mark Kilgard在SGI时写的。作为AUX库的功能更强大的替代品，用于隐藏不同窗口系统API的复杂性。是一个学习OpenGL编程的一个良好开端。.

1、安装glut

1）、首先下面win下的glut，然后解压，地址如下
http://www.opengl.org/resources/libraries/glut/glutdlls37beta.zipk
2）、把glut.h头文件放到你的vs/vc/include/GL的目录下，如果没有GL目录，则创建一个，如我的vs目录是D:\vs2012\VC\include\GL

3）、把解压得到的glut.lib和glut32.lib放到静态函数库所在文件夹（即与include并排的lib文件夹下）。
4）、把解压得到的glut.dll和glut32.dll放到操作系统目录下面的system32文件夹内。（典型的位置为：C:\Windows\System32，如果是64位的话则放在C:\Windows\SysWOW64）

2、创建vs工程

选择File->New->Project，然后选择Win32 ConsoleApplication,选择一个名字，然后按OK。在谈出的对话框左边点Application Settings，找到Empty project并勾上，选择Finish。然后向该工程添加一个代码文件，取名为“OpenGL.cpp”
在OpenGL.cpp中包含头文件

也可在工程属性->配置属性->连接器->命令行，的附加选项中加入上述三个lib文件

添加main.cpp并编写代码

#include <GL/glut.h>
void myDisplay(void)
{
    glClear(GL_COLOR_BUFFER_BIT);
    glRectf(-0.5f, -0.5f, 0.5f, 0.5f);
    glFlush();
}
int main(int argc, char *argv[])
{
    glutInit(&argc, argv);
    glutInitDisplayMode(GLUT_RGB | GLUT_SINGLE);
    glutInitWindowPosition(100, 100);
    glutInitWindowSize(400, 400);
    glutCreateWindow("第一个OpenGL程序");
    glutDisplayFunc(&myDisplay);
    glutMainLoop();
    return 0;
}
